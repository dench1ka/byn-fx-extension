## Что это за проект

Chrome-расширение для жителей и покупателей Беларуси. Автоматически конвертирует цены в белорусских рублях (BYN) в доллары, евро или рубли прямо на сайтах av.by и kufar.by — по официальному курсу Нацбанка РБ. Никакой ручной работы: открыл страницу, рядом с каждой ценой появился компактный синий бейдж с суммой в нужной валюте.

**Целевая аудитория:**
- Покупатели автомобилей на av.by и kufar.by (основная)
- Покупатели недвижимости на kufar.by
- Покупатели товаров на kufar.by
- Люди из России, смотрящие недвижимость/авто в Беларуси (хотят видеть в рублях)
- Белорусы, ориентирующиеся на доллар при оценке стоимости

**Контекст появления:** До 2024 года av.by и kufar.by сами показывали цены в долларах. МАРТ (Министерство антимонопольного регулирования) выдало предписания убрать цены в иностранной валюте — ссылаясь на Стратегию повышения доверия к национальной валюте до 2035 года. Сайты убрали. Расширение восстанавливает эту функцию на стороне браузера пользователя.

**Правовой статус:** Законно. Предписания МАРТ адресованы владельцам сайтов, а не пользователям. Расширение работает локально в браузере, не изменяет сайт, не занимается торговлей. Аналог — переводчик страниц или тёмная тема. Chrome Web Store — американская платформа вне юрисдикции МАРТ. В описании расширения указано: «Справочный инструмент для личного пользования. Конвертация носит справочный характер по курсу Нацбанка РБ».

## Функциональность

### На сайтах (content.js)

**Автоматическая конвертация цен:**
- Рядом с каждой BYN-ценой появляется компактный синий бейдж
- Формат: `$13,850` / `€12,500` / `₽5.1М`
- Бейдж: `display: inline`, вставляется через TreeWalker сразу после числа цены — не падает на новую строку вне зависимости от flex/block родителя

**Поддерживаемые сайты:**
- `av.by` и все поддомены (`cars.av.by`, `parts.av.by` и др.)
- `kufar.by` и все поддомены (`auto.kufar.by`, `re.kufar.by` и др.)

**Двойной механизм поиска цен:**
1. **CSS-селекторы** (быстро) — список из ~20 точных селекторов для av.by и kufar.by
2. **Leaf-scanner** (универсальный fallback) — TreeWalker обходит DOM и ищет маленькие элементы с BYN-ценой. Работает на любом SPA с CSS-modules (классы вида `price__mTq3Z`), не зависит от имён классов

**Что фильтруется (не конвертируется):**
- Пробег: `97 000 км`, `444 444 км`
- Единицы измерения: `128 см`, `150 Вт`, `256 ГБ`, `120x200`
- Даты и время: `09:34`, `14 мар., 09:34`, `21 мая, чт`
- Годы без суффикса: `1990`, `2021`
- Количества объявлений: `357`, `42` (без суффикса)
- Технические числа: числа выше 800 000 BYN на av.by, выше 600 000 BYN на kufar.by

**Форматирование бейджа по валютам:**
- USD/EUR: `$13,850` / `€12,500` (en-US локаль)
- RUB до 1М: `₽142 857` (полное число с пробелом)
- RUB от 1М: `₽5.1М` (сокращение — не влезает в контейнеры кufar.by)

**Реакция на изменение настроек:** `chrome.storage.onChanged` — при смене целевой валюты все бейджи на текущей странице моментально перерисовываются без перезагрузки.

**MutationObserver** с debounce 280мс — подхватывает динамически подгружаемые карточки (SPA-навигация, бесконечный скролл). Игнорирует собственные мутации (вставку бейджей) чтобы не зациклиться.

### Курсы валют

**Источник:** Официальный API Нацбанка РБ: `https://api.nbrb.by/exrates/rates?periodicity=0`

**Один запрос** возвращает все курсы (USD, EUR, RUB, PLN, CNY, GBP). Раньше было 7 параллельных запросов.

**Кеширование:** 1 час в `chrome.storage.local`. При недоступности API (VPN, блокировки) — берёт устаревший кеш или fallback-курсы.

**Динамика курсов:** Отдельный endpoint `/exrates/rates/dynamics/{Cur_ID}?startDate=...&endDate=...`. Кеш графиков: 6 часов.

**VPN:** NBRB API заблокирован в России — это известная особенность, не баг. При недоступности API показывается метка "нет связи · резервные курсы", расширение продолжает работать с кешированными данными.

### Popup (интерфейс расширения)

**Дизайн:** Синий акцент (`#60a5fa` тёмная тема, `#2563eb` светлая), нейтральные чёрно-белые фоны без цветных заливок. Стиль в духе Linear/Vercel.

**Три темы:** Авто (следует за `prefers-color-scheme`), Тёмная, Светлая. Выбор сохраняется в `chrome.storage.local`. Слушает системное изменение темы в реальном времени.

**Hero-блок:** Большой курс USD с дельтой к предыдущему обновлению (↑ +0.18%).

**Конвертер валют:**
- `<input type="text" inputmode="decimal">` — не вызывает ошибку Chrome "cannot be parsed" (была с `type="number"`)
- Поддерживает ввод с запятой и точкой: `262,12` = `262.12`
- Кнопка swap ⇅ (переворачивает валюты с анимацией)
- 7 валют: BYN, USD, EUR, RUB, PLN, CNY, GBP

**Сетка курсов:** 6 карточек (USD, EUR, RUB·100, PLN, CNY·10, GBP) с дельтой. Клик → открывает график.

**Графики курсов:**
- Табы: 7Д / 1М / 3М / 1Г
- SVG line chart с градиентной заливкой, цвет линии меняется (синий/красный) по направлению
- Статистика: мин, макс, изменение %

**Настройки (slide-in панель, кнопка ⚙):**
- Тема: Авто / Светлая / Тёмная
- Включение/выключение каждого сайта отдельно (av.by, kufar.by)
- **Целевая валюта бейджей на сайтах:** USD / EUR / RUB (три кнопки)
- Показывать дельту курса ↑/↓

**Нет inline onclick** — полностью CSP-compliant для Manifest V3. Все обработчики через `addEventListener`.

## Известные ограничения

1. **VPN + РФ:** NBRB API заблокирован в России. Работает с кешем/fallback.
2. **Новые классы Куфара:** При очередном деплое CSS-module хеши меняются. Leaf-scanner (TreeWalker) это нивелирует, но если цена вдруг в нетипичном контейнере — может не найти.
3. **Страницы бронирования Куфара:** Могут содержать специфичные числа. Фильтры дат и верхний порог BYN снизили ложные срабатывания, но не устранили полностью.
4. **realt.by:** Убран из поддержки — на сайте есть собственные долларовые цены, конфликт с бейджами.
5. **Нет поддержки Firefox** — только Chrome/Edge/Brave (Manifest V3).

## Монетизация — варианты

### 1. Донаты (старт, нулевые затраты)
Кнопка "Поддержать" в настройках → Buy Me a Coffee / ROBOKASSA (для РБ).
Конверсия ~0.5-2% активных пользователей. При 5 000 пользователей = 25-100 доноров/мес.

### 2. Разовая покупка Pro-версии ($1-3)
Базовая версия бесплатно, Pro-фичи за разовый платёж:
- История просмотров
- Уведомления о ценах
- Расширенные графики
Платформа: Stripe + Gumroad, или через Chrome Web Store In-App Payments.

### 3. Партнёрство с автодилерами/агрегаторами (долгосрочно)
При аудитории 10 000+ пользователей — переговоры с автосалонами о нативной рекламе в popup (НЕ баннеры, а рекомендации). Например: "Похожие авто со скидкой у партнёра X".

### 4. B2B API (очень долгосрочно)
Если вырастет до крупного инструмента — API для сайтов которые хотят показывать конвертацию на своей стороне легально.
